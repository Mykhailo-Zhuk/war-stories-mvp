# 📖 War Stories — Історії з війни

**Проєкт:** OPE-6 — "Створити проект Історії з війни"  
**Статус:** ✅ Повністю реалізовано  
**Технології:** Next.js 14, TypeScript, Tailwind CSS, Linear Design  
**Зберігання:** localStorage (клієнтське)

## 📋 Функціонал

### ✅ Реалізовано:
1. **📝 Написання історій** — створення нових історій через модальне вікно
2. **👀 Перегляд історій** — сітка карток з прев'ю, детальний перегляд у модалці
3. **💾 Зберігання історій** — автоматичне збереження в localStorage
4. **🔐 Шифрування даних** — захист конфіденційних історій
5. **🔍 Пошук історій** — за назвою, автором, текстом
6. **📂 Категорії** — Історії, Листи, Звіти, Вірші, Спогади
7. **❤️ Вподобання** — лайки до історій
8. **🗑️ Видалення** — можливість видаляти історії
9. **📊 Статистика** — загальна кількість історій, вподобань, переглядів
10. **📱 Mobile-first** — адаптивний дизайн

### 🎨 Дизайн:
- **Темна тема Linear-стиль** — кольори: `#0d1117`, `#c9d1d9`, `#1f6feb`, `#30363d`
- **Адаптивна сітка** — 1/2/3 колонки залежно від екрану
- **Модальні вікна** — створення та перегляд історій
