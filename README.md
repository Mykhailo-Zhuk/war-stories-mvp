# 📖 War Stories MVP — Зберігання та поширення історій про війну

**Проєкт:** OPE-6 — "Створити MVP для зберігання та поширення історій про війну"  
**Статус:** ✅ Повністю реалізовано (MVP)  
**Технології:** Next.js 14, TypeScript, Tailwind CSS, **Zhuk Design System**

## 🎨 **Zhuk Design System - Новий дизайн**

Проєкт повністю перероблено з використанням **Zhuk Design System** - стандартної дизайн-системи для всіх проектів Misha Zhuk, створеної на основі Linear Dark Theme.

### 🎯 **Ключові принципи:**
- **Технічний мінімалізм** - функціональність понад декор
- **Темна тема стандарт** - фон `#1a1a1a`, текст `#ffffff` → `#c9c9c9` → `#888888`
- **Чітка ієрархія** - типографіка за рівнями (28px → 11px)
- **Система відступів** - 4px, 8px, 16px, 24px, 32px, 48px
- **Консистентні компоненти** - однакові розміри, кольори, радіуси

### 🎨 **Кольорова палітра:**
- **Фон:** `#1a1a1a` → `#1e1e1e` → `#2a2a2a`
- **Текст:** `#ffffff` → `#c9c9c9` → `#888888` → `#666666`
- **Акцент:** `#7C5CFC` (фіолетово-синій)
- **Статуси:** Done `#4CAF50`, In Progress `#F5A623`, Urgent `#E5484D`
- **Поточна сесія:** `#4ADE80` (яскраво-зелений)

### 📦 **Компоненти дизайн-системи:**
1. **Картки історій** з категоріями та статусами
2. **Бічна панель** налаштувань (220px, активний стан з підсвіткою `#2a2a2a`)
3. **Хлібні крихти** (11px, сірий, формат `Parent › Child`)
4. **Статусні індикатори** (12px кружечки: new=🆕, important=⚠️, later=⏰)
5. **Аватари** (18px, 20px, 32px, з ініціалами)
6. **Панель властивостей** (220px, dropdown мітки)
7. **Перемикачі** (40×22px, анімовані)
8. **Прогрес бар** (0/6 з іконкою checkbox)
